# 💸 Expense Tracker - GenZ Style Web App

## 🎯 Project Overview
เว็บแอปบันทึกรายรับรายจ่าย สไตล์วัยรุ่น GenZ สวยงาม ทันสมัย ใช้งานง่าย

---

## 🛠️ Skills & Technologies Required

### Frontend (Primary Focus)
- **HTML5** - Structure & Semantic Markup
- **CSS3** - Styling, Animations, Flexbox/Grid
- **JavaScript (ES6+)** - Interactivity & Logic
- **Responsive Design** - Mobile-first approach

### Recommended Frameworks/Libraries
- **React.js** หรือ **Vue.js** (เลือกอย่างใดอย่างหนึ่ง)
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** หรือ **GSAP** - Animations
- **React Router** / **Vue Router** - Navigation

### Design Tools
- **Figma** - UI/UX Design & Prototyping
- **Coolors.co** - Color Palette Generation
- **Google Fonts** - Typography

### Backend (Future Implementation)
- **Node.js** + **Express** หรือ **Python** + **FastAPI**
- **MongoDB** หรือ **PostgreSQL** - Database
- **JWT** - Authentication
- **bcrypt** - Password Hashing

### Version Control
- **Git** & **GitHub** - Code Management
